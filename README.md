# 🐰 Stanford Bunny Viewer

基于 OpenGL 3.3 的 Stanford Bunny 3D 模型查看器，支持实时交互、顶点拾取和模型变换。

---

## 📋 功能特性

- ✅ **3D 模型渲染**：加载 PLY2 格式的 Stanford Bunny 模型
- ✅ **第一人称相机**：WASD 移动 + 鼠标视角控制
- ✅ **模型变换**：平移、旋转、缩放
- ✅ **顶点拾取**：鼠标点击选取顶点，显示 3D 坐标和法线
- ✅ **可视化标记**：小立方体高亮显示选中的顶点
- ✅ **双模式渲染**：实体/线框模式切换
- ✅ **背面剔除优化**：只渲染朝向相机的顶点

## 🎮 操作说明

### 相机控制

| 按键 | 功能 |
|------|------|
| `W` | 相机向前移动 |
| `A` | 相机向左移动 |
| `S` | 相机向后移动 |
| `D` | 相机向右移动 |
| `TAB` | 切换鼠标锁定/自由（切换渲染模式） |
| 鼠标滚轮 | 调节视野（Zoom） |

| 按键 | 功能 |
|------|------|
| `←` / `→` | 兔子左右平移 |
| `J` / `L` | 兔子绕 Z 轴旋转 |
| `I` / `K` | 兔子绕 X 轴翻转 |
| `↑` / `↓` | 兔子放大/缩小 |

### 拾取功能

| 操作 | 功能 |
|------|------|
| **鼠标左键** | **点击选取顶点** |
| `P` | 显示/隐藏拾取点标记 |
| `ESC` | 退出程序 |

## 📊 技术细节

### 渲染模式

- **实体模式**（光标隐藏）：`glPolygonMode(GL_FILL)`
- **线框模式**（光标显示）：`glPolygonMode(GL_LINE)`

### 顶点拾取算法

1. 获取鼠标屏幕坐标
2. 遍历所有顶点，投影到屏幕空间
3. 计算屏幕距离，选择最近的顶点
4. 阈值判断（40 像素半径）
5. 输出顶点信息（索引、3D 坐标、法线）

### 标记渲染

- 使用小立方体（边长 0.01 单位）
- 始终实体渲染（不受 TAB 影响）
- 位置：选中顶点的世界坐标
- 着色器：`select.vs` / `select.fs`

## 📝 使用说明

### 查看模型

1. 运行程序后，默认显示实体渲染的兔子模型
2. 使用 WASD 移动相机，观察模型不同角度
3. 按 TAB 切换到线框模式，查看模型拓扑结构

### 选取顶点

1. 按 TAB 切换到线框模式（显示鼠标）
2. 移动鼠标到目标顶点
3. 点击鼠标左键选取顶点
4. 命令行输出顶点信息：
   ```
   ✅ 成功选中点！
      索引号：2567
      3D 坐标：(0.234, -0.567, 0.891)
      法线向量：(0.123, 0.456, 0.789)
   ```
5. 小立方体标记会出现在选中的顶点位置
6. 按 P 键可以开关标记显示

### 调整模型位置

- 使用方向键平移模型
- 使用 J/L 键旋转模型
- 使用 I/K 键翻转模型
- 使用 ↑/↓ 键缩放模型

## 🎨 着色器

### bunny.vs / bunny.fs

### select.vs / select.fs

用于渲染顶点标记立方体的着色器，使用醒目的颜色高亮显示。
