# Bunny Viewer 使用说明

## 运行
```
.\output\main.exe
```

## 操作

| 按键 | 功能 |
|------|------|
| W/A/S/D | 相机移动 |
| 方向键 ←→ | 兔子平移 |
| J/L | 兔子旋转 |
| I/K | 兔子前后翻转 |
| ↑↓/滚轮 | 缩放 |
| TAB | 切换视角控制 |
| **鼠标左键** | **拾取顶点** |
| P | 显示/隐藏拾取点 |
| ESC | 退出 |

## 依赖
- C:\mingw64\bin 在 PATH 中
- `bunny_normal.ply2` 和 `glfw3.dll` 在项目根目录
